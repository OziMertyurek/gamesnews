# GamesNews — Proje Planı ve Teknik Doküman

## 0. Proje Özeti

**GamesNews**, oyun haberleri, platformlara göre ürün listeleri (fiziksel + dijital), dijital mağaza indirimleri ve oyun türlerine göre keşif sayfaları sunan bir web uygulamasıdır.

Amaç:

* Türkiye odaklı fiyat bilgisi
* Güncel indirimler
* Oyun bilgileri ve oyuncu sayıları
* Modern, hızlı ve ölçeklenebilir frontend mimarisi

---

# 1. Teknoloji Stack

## Frontend

* Vite
* React
* TypeScript
* TailwindCSS
* React Router
* TanStack Query (data fetching + cache)

## Veri Kaynakları

### Steam

* Fiyat + indirim (TRY):
  `store.steampowered.com/api/appdetails?cc=tr`
* Oyuncu sayısı:
  `ISteamUserStats/GetNumberOfCurrentPlayers`

### Epic & Xbox

MVP:

* Manuel veri (JSON seed)

Faz-2:

* 3. parti API veya veri sağlayıcı

### Fiziksel ürün fiyatları (TR)

MVP:

* Manuel seed verisi

Faz-2:

* Affiliate feed / resmi API (varsa)
